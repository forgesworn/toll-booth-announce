# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in toll-booth-announce, please report it responsibly via [GitHub Security Advisories](https://github.com/forgesworn/toll-booth-announce/security/advisories/new).

Do **not** open a public issue for security vulnerabilities.

## Scope

toll-booth-announce handles Nostr secret keys for signing announcement events. If you find an issue related to key handling, event signing, or any other security-sensitive behaviour, please report it immediately.
